# 🚀 CRUD Application (Laravel + MySQL)

A professional Role-Based CRUD (Create, Read, Update, Delete) web application built with Laravel and MySQL.  
This project demonstrates secure authentication, role-based access control, and a clean dashboard interface for managing records efficiently.

---

## 📌 Features

- 🔐 User Authentication (Login / Register / Logout)
- 👥 Role-Based Access Control (Admin / User)
- ➕ Create new records
- 📖 View all records in a structured table
- ✏️ Edit and update records
- ❌ Delete records securely
- 🔍 Search functionality
- 📄 Pagination for large datasets
- 📊 Simple dashboard with summary stats
- 🧾 Activity tracking (optional enhancement)
- ⚡ Form validation & error handling

---

## 🛠️ Tech Stack

- **Backend:** Laravel (PHP Framework)
- **Frontend:** Blade Templates / Bootstrap 5
- **Database:** MySQL
- **Authentication:** Laravel Breeze / Built-in Auth
- **Version Control:** Git & GitHub
